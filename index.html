BigInt JavaScript Library

By Robin Hu

Just thought I'd implement a quick BigInt library in JavaScript. There's addition, subtraction and multiplication supported. No division for now, as I am interested in using this to calculate large integers.

You can view the source code here: https://github.com/robinfhu/robinbigint

Algorithm's Used

  • Addition: simply arithmetic add and carry. Taught in grade school.
  • Subtraction: complement method. http://en.wikipedia.org/wiki/Method_of_complements.
  • Multiplication: http://en.wikipedia.org/wiki/Karatsuba_algorithm
